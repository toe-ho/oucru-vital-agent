## Brand In One Line

> **Vital Agent** — know which signals you can trust, in minutes, with the
> evidence behind every call.

- **Visual direction:** "Instrument" — precision, calm, engineered honesty.
- **Voice model:** warm shell (marketing/docs) + neutral core (verdict surfaces).
- **Positioning:** standalone product; OUCRU credited as origin.

## Non-Negotiables (read before any brand change)

1. **Status colors are sacred.** Emerald (accept) / red (reject) / amber
   (uncomputable, stale) carry *clinical meaning*. Never reuse them as brand or
   decorative colors.
2. **One identity hue.** Brand Indigo is the single primary + interactive color.
   No stray legacy blue-500.
3. **Verdict surfaces lead with the classification + the number.** Never with
   adjectives ("bad signal").
4. **The chatbot never gives clinical advice.** It is a grounded analyst that
   cites evidence and refuses ungrounded claims.
5. **Waveform imagery stays abstract** — must never resemble a real diagnostic
   trace implying a clinical reading.
